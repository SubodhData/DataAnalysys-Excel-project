Food Delivery Sales Analysis Dashboard

• Developed an interactive Power BI dashboard to analyze food delivery transaction data and uncover insights related to customer behavior, product demand, and revenue distribution.

• Performed data cleaning and transformation using Power Query, including handling missing values, correcting data types, and standardizing date and numerical formats.

• Built a data model and created DAX measures to calculate key metrics such as total bill amount, order quantity, and customer-level revenue.

• Designed a business-focused dashboard to help stakeholders monitor sales performance, customer activity, and product trends.
